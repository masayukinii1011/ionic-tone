# ToneSynth
IonicとTone.js(Web Audio APIを扱うライブラリ)でシンセサイザーを作成しました。  
Androidアプリ、デスクトップアプリ(Electron)、PWAとして使用できます。  
4x4のパッドに触れると音を出力。  
スライダーでエフェクトをかけられます。
